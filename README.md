# pipedrive-app

### build
```console
docker build -t yedaysal/pipedrive-app:1.0 .
```

### pull
```console
docker pull docker.io/yedaysal/pipedrive-app:1.0
```

### run
```console
docker run -d -p 8000:8080 --name pipedrive-app yedaysal/pipedrive-app:1.0
```

### exec
```console
docker exec -it pipedrive-app /bin/bash
```
